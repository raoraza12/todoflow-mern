# ⚡ TodoFlow — MERN Stack Todo App

> **Powered by Supabase (PostgreSQL)** 　

A full-stack Todo app -- **React, Express, Node.js, Supabase** -
- push notifications, dark UI, PWA-ready.

> 🚀 [Deploy Frontend to Vercel](https://vercel.com/import)   |  📧 [Deploy Backend to Render](https://render.com/deploy)

## 🚀 Quick Start

```bash
git clone https://github.com/raoraza12/todoflow-mern.git
cd todoflow-mern
npm run install:all
npm run dev
```

- **Frontend** : http://localhost:3000
- `client/` - deploy this to Vercel
- **Backend** : http://localhost:5000
- `server/` - deploy this to Render/Railway

## 📍 Supabase Setup

1. Go to [supabase.com](https://supabase.com) → Create project
2. Settings → API → Copy URL + anon key
3. Paste into `server/.env`

## ✅ Features

- Full CRUD with Supabase PostgreSQL
- Push Notifications (works on mobile!)
- Live Stats -- Total, To Do, In Progress, Done
- Search, Filter, Sort
- Dark, beautiful UI with animations
- PWA-ready, mobile-first
- Due Dates, Categories, Priorities
